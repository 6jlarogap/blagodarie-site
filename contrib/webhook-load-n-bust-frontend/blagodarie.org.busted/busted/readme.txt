Здесь в папках dev/<время>,  prod/<время> лежат каталоги веток проекта благодари.рф,
с index.html's, в которых изменены hrefs, указывающие на js/css.

Изменены они командой grunt.
Установка grunt:

sudo apt install npm node-grunt-cli
# В этой папке:
npm init
npm install grunt --save-dev
npm install grunt-cli --save-dev
npm install grunt-cache-bust --save-dev
