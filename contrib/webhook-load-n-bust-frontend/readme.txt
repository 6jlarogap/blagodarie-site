Задача: 
    при всяком коммите из любой ветки проекта:
        - обновлять содержимое documentRoot версии проекта, соответствующей ветке гита.
        - чтоб при изменении js/css файлов это сразу
          отражалось у клиента, а не хватало старые js/css
          из кэша браузера.
          Т.е. изменять там в index.html's фронта ссылки на js/css файлы,
          например, чтоб 

            <script src="settings.js"></script> заменялся на
            <script src="settings.e8fa8ead84442467.js"></script>

            где e8fa8ead84442467 - md5 hash содержимого settings.js
        
          При этом js/css файлы не перименовываем!
          Посредством настроек виртуальных хостов апача делаем так,
          что при обращении, например, к settings.e8fa8ead84442467.js
          клиенту отдавался settings.js

Подробнее см. webhook-scrips/blagodarie.org/readme.txt
